# РГР №1: Чисельне диференціювання функцій
**Виконала:** Нежива Юлія Дмитрівна, група КБ-21
**Навчальний заклад:** ХНУ імені В. Н. Каразіна

## Опис проєкту
Проєкт реалізує об'єктно-орієнтований застосунок для розрахунку похідної аналітичних та табличних функцій.

### Функціонал:
- Обчислення похідної методом центральної різниці.
- Робота з файлами .dat через GUI.
- Візуалізація результатів у вигляді таблиць та графіків.

### Технології:
- Java 23 (JDK 23)
- Swing (для графічного інтерфейсу)
- Git (контроль версій)
